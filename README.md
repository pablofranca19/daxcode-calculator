# Dax Code - Age Calculator

A simple calculator that figures out your age from the birth year you enter.

![Dax Code logo](assets/images/logo-projeto.png)

## ✨ Features

- Enter your birth year
- Click "Calculate" and see your age instantly
- Simple, direct interface — no page reload

## 🛠️ Tech stack

- HTML
- CSS
- JavaScript (vanilla)

## 🚀 How to run

This is a static project with no dependencies or build steps.

1. Clone the repository
   ```bash
   git clone https://github.com/pablofranca19/daxcode-calculator.git
   ```
2. Open `index.html` in your browser

That's it — no server or installation needed.
